# Parceiros Sugeridos para Anúncios Colaborativos (Business Collaborative Ads Suggested Partners)

Recupera parceiros sugeridos de Anúncios Colaborativos para um negócio.

## Leitura
**Endpoint:** `GET /{business_id}/collaborative_ads_suggested_partners`

### Campos Retornados
A resposta contém uma lista de nós `CPASAdvertiserPartnershipRecommendation`.

### Resposta
```json
{
    "data": [],
    "paging": {}
}
```

## Operações Não Suportadas
*   **Criação:** Não suportado neste endpoint.
*   **Atualização:** Não suportado neste endpoint.
*   **Exclusão:** Não suportado neste endpoint.

## Códigos de Erro Comuns
| Código | Descrição |
| :--- | :--- |
| **100** | Parâmetro inválido. |
