# Anúncios de Mensagem (Visão Geral)

Direcione usuários para conversas no Messenger, Instagram ou WhatsApp.

## Tipos de Anúncio
1. **Clique para o Messenger:** Inicia uma conversa direta.
2. **Geração de Leads (Messenger):** Usa um fluxo automatizado de perguntas e respostas para qualificar leads.
3. **Extensões de Produto:** Exibe produtos do catálogo abaixo da mídia principal (Otimização Advantage+).
4. **Clique para Assinar (Marketing Messages):** Convida o usuário a optar por receber mensagens recorrentes (ex: promoções diárias).

## Benefícios
- Atendimento individualizado.
- Automação via Chatbots/Fluxos.
- Escala de comunicação direta.
